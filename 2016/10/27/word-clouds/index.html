Generating Wordclouds from Collections of Reviews

Using WhiskyMetrics to perform elementary text analytics.

Posted by Neelabh Kashyap on October 27, 2016

Creating a Word Cloud from Whisky Reviews

Connect to the database

%matplotlib inline

import dbconfig,psycopg2,wordcloud

# Connection
conn = psycopg2.connect(database=dbconfig.database,
                        host=dbconfig.host,
                        port=dbconfig.port,
                        user=dbconfig.user,
                        password=dbconfig.password)

# Cursor
c = conn.cursor()

# Test
c.execute('select count(*) from metadata;')

[m for m in c]

[(731,)]

This looks correct.

Get all reviews for Laphroaig 10

c.execute("select review_text from review where review_id in (select review_id from metadata where topic='laphroaig 10');")

text = ' '.join((m[0] for m in c))

Create and display a word-cloud

import matplotlib.pyplot as plt 

wc = wordcloud.WordCloud(stopwords=set(['laphroaig', '10','color','nose','taste','finish']),
                       max_words=1000).generate(text)

default_colors = wc.to_array()
plt.title("Default colors")
plt.imshow(default_colors)
plt.axis("off")
plt.show()

No surprises there! ;)

Let’s look at the Laphroiag Quarter Cask now and see if it’s any different…
